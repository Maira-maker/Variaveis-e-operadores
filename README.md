# Variaveis-e-operadores
Este repositório contém três códigos em Python que resolvem problemas matemáticos usando operadores. O objetivo é praticar lógica de programação.

cotacao_real: declarei variáveis para armazenar os valores da cotação do dólar, euro, franco suiço e won sul-coreano, implementando uma lógica de cálculo para converter reais nessas moedas. Referência 09/05/2026

desconto_5%: função que recebe o valor original de um produto e aplica um cálculo para subtrair 5%, mostrando o valor inicial e o valor final com desconto.

quantidade_tinta: utiliza variáveis para armazenar os valores de largura e altura da parede, e aplica operadores matemáticos como multiplicação e divisão para calcular a área, dividindo por dois para a quantidade de tinta necessária.
